## 🚀 快速开始

``````bash
# 克隆项目
git clone https://github.com/fawangyyds/my_home.git
``````

## 📄 本地使用指南

**在浏览器设置-->启动、主页和新建选项卡页-->启动时-->打开自定义网站-->添加站点-->本地index.html的路径（file:///D:/个人主页/index.html）**
